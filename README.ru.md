# Friends Bot

Телеграм-бот для ежедневных розыгрышей "Красавчик дня" и "Пидор дня", вдохновленный репозиторием [TheUserOfTheDayBot](https://github.com/DevDmitryN/TheUserOfTheDayBot), но переписанный на Python с использованием современного стека.

## Особенности

- **Стек:** Python 3.12+, `aiogram 3.x`, `SQLite`.
- **Управление:** Использование `uv` для менеджмента зависимостей и виртуального окружения.
- **Надежность:** Статическая типизация `mypy`, модульные тесты `pytest` и логирование.
- **Безопасность:** Ограничение доступа по `ALLOWED_CHAT_ID` (защита от использования в посторонних группах).
- **Асинхронность:** Двухуровая защита от возможного состояния гонки (уровень кода и уровень БД).
- **Архитектура:** Чистая модульная структура кода с разделением на обработчики, базу данных и конфиг.

## Быстрый старт

### 1. Настройка окружения

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_от_botfather
DB_PATH=friends_bot.db
ALLOWED_CHAT_ID=id_вашей_telegram_группы
```

### 2. Установка и запуск

Проект использует `Makefile` для удобства:

```bash
make install    # Установка dev-зависимостей
make run        # Запуск бота
make test       # Запуск тестов
make type       # Проверка типов
make lint       # Проверка стиля кода
make format     # Автоформатирование кода
make clean      # Очистка кэша и окружения
```

## Команды

- `/reg` – регистрация в игре.
- `/run` – запуск розыгрыша "Красавчик дня".
- `/pidor` – запуск розыгрыша "Пидор дня".
- `/stats` – просмотр статистики побед.
- `/pidorstats` – просмотр статистики поражений.
- `/delete` – выход из игры (с сохранением истории в БД).
